# FTW

Find The Word (FTW) is a game where player have to find correct word.

# HOW TO INSTALL AND RUN ?


## PREREQUISITES ( Please verify if you have installed )

* Python3

## (ON LINUX)


```
git clone https://github.com/N0vachr0n0/FindTheWord_Game.git
cd FindTheWord_Game
python3 FTW.py
```

Enjoy it !
